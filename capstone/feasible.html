Feasibility Plan: Future Light Pollution Visualization

Planned Features

  • Timeline of light source evolution (with illustrations)
  • Side-by-side radiance comparison visual for each light type
  • Interactive slider to simulate sky visibility over time
  • Animated lighting effect layers (e.g. glow, skyglow)
  • Responsive layout for mobile and desktop

Technologies Required

  • HTML & CSS – for structure and layout
  • JavaScript – for slider interactivity and DOM control
  • p5.js or Three.js – for radiance visualization and glow effects
  • SVG / PNG illustrations – for static light source visuals
  • Figma – for wireframes and component mockups

Skill Assessment

  • HTML/CSS: Confident and experienced
  • JavaScript: Comfortable with DOM and interactivity basics
  • p5.js: Intermediate – can build basic animations and visual effects
  • Three.js: Limited experience – may substitute with p5.js for feasibility
  • SVG editing: Proficient using Illustrator or Figma for asset prep
  • Figma: Fluent in creating UI layouts and prototypes
